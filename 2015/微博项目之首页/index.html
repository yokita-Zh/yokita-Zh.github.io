微博项目之首页

微博-首页显示微博列表

微博首页是浏览微博列表的页面，也是本次微博项目最主要的功能之一，下面列举一下涉及到的各种知识。

  • 首先网络请求数据

网络请求数据是在登录成功后进行的，在登录成功后会得到一些参数比较重要的是access_token，之后请求数据的时候用这些参数来判断是否登录成功，然后发给首页通知可以请求微博数据了。

  • 模型类的设计

//2.转换成可以解析的格式
NSData statusData = [contentStr dataUsingEncoding:NSUTF8StringEncoding ];

NSArray status = statusDtic[@”statuses”];
//可变数组，保存创建的model
NSMutableArray mutaStatusArray = [NSMutableArray arrayWithCapacity:status.count];
//每个数据一次绑定
for (NSDictionary staDitc in status) {
StatusModel statusModel = [[StatusModel alloc] initWithDictionary:staDitc];

}
